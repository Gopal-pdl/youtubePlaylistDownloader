# \# YouTube Playlist Downloader

# 

# A Windows Forms application for downloading YouTube playlists as MP3 files.  

# Built with .NET Framework 4.8 and \[YoutubeExplode](https://github.com/Tyrrrz/YoutubeExplode).

# 

# \## Features

# 

# \- Download audio from YouTube playlists in MP3 format

# \- Select download folder

# \- Limit number of downloads per playlist

# \- Skip files with similar names (≥50% word match) to avoid duplicates

# \- Review skipped files and decide later

# \- Stop downloads at any time

# \- Modern, user-friendly interface

# 

# \## Requirements

# 

# \- \*\*.NET Framework 4.8\*\*

# \- \*\*FFmpeg\*\*  

# &nbsp; Download \[ffmpeg.exe](https://ffmpeg.org/download.html) and place it in the application directory.

# \- \*\*NuGet Packages\*\*  

# &nbsp; - \[YoutubeExplode](https://www.nuget.org/packages/YoutubeExplode)

# 

# \## Setup

# 

# 1\. \*\*Clone or download this repository.\*\*

# 2\. \*\*Open the solution in Visual Studio 2022.\*\*

# 3\. \*\*Install NuGet dependencies:\*\*

# &nbsp;  - Right-click the project > Manage NuGet Packages > Install `YoutubeExplode`

# 4\. \*\*Download FFmpeg:\*\*

# &nbsp;  - Place `ffmpeg.exe` in the output directory (e.g., `bin\\Debug`).

# 5\. \*\*Build and run the project.\*\*

# 

# \## Usage

# 

# 1\. \*\*Enter the YouTube playlist URL.\*\*

# 2\. \*\*Choose the download folder.\*\*

# 3\. \*\*Set the number of downloads (optional).\*\*

# 4\. \*\*Click "Download Playlist as MP3".\*\*

# 5\. \*\*Monitor progress and review skipped files.\*\*

# 6\. \*\*Click "Stop" to cancel ongoing downloads.\*\*
